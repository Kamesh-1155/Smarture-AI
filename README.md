# 🌱 Smarture - AI Smart Agriculture System

## Description
AI-based smart agriculture system that predicts the best crop using soil and weather data and recommends the most profitable crop.

## Features
- Crop Prediction using Machine Learning
- AI Recommendation System
- Profit-based Ranking
- Explainable Output

## Tech Stack
- Python
- Scikit-learn
- Streamlit

## Run
- pip install -r requirements.txt
- python model.py
- streamlit run app.py
